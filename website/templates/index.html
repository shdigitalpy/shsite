Was ist Django

Django ist ein High-Level Python Web Framework was schnelles und sauberes Entwickeln ermöglicht. Python hat sich als trendigste Programmiersprache im 2020 herausgestellt, dieser Trend setzt sich auch im 2021 fort. Django als Teil von Python wurde von erfahrenen Webentwicklern ins Leben gerufen und erledigt einen Großteil der Probleme bei der Webentwicklung, sodass der Entwickler sich auf das Konzipieren der Webseite oder App konzentieren kann, ohne das Rad neu erfinden zu müssen. Es ist kostenlos, Open Source und lächerlich schnell.


Was ist React

React ist eine JavaScript Softwarebibliothek, welche sich sowohl für den Aufbau komplexer Anwendungen sowie für Single-Page-Webanwendungen eignet. React wurde 2011 von Facebook entwickelt und später im 2012 für Instagram eingesetzt. Bei moderner Webentwicklung im Frontend Bereich darf React nicht mehr fehlen.
